## What it is

A personal blog with no ads, no trackers, no noise. Just writing.

Built with Astro for fast static pages, styled with warm typography, and managed through Decap CMS so I can write from any browser.

## Why I built it

I wanted a space that felt like mine — not Medium, not Substack, not someone else's platform with someone else's rules.

## Stack

- **Astro** — static site generation
- **Decap CMS** — browser-based content editor
- **GitHub Pages** — free hosting with auto-deploy
- **Lora + DM Sans** — the font pairing I keep coming back to
